      How It Works

      현재 엔진이 점수를 계산하는 방식

      매수 점수 규칙
      • `EMA20 > EMA50` 이면 +8
      • `EMA50 > SMA200` 이면 +8
      • 종가가 `EMA20` 위면 +5, `SMA200` 위면 +5
      • `ADX > 25` 이고 `+DI > -DI` 이면 +4
      • MACD 골든크로스면 +8, 히스토그램 개선이면 +4
      • `RSI 45~65` 이면 +6, 과매도 반등이면 +7, `RSI > 75` 이면 -6
      • 거래량 비율이 `1.5x` 이상이면 +8, 상승일에 `1.2x` 이상이면 +5
      • OBV 상승이면 +4, 저항 돌파와 거래량 급증 동반이면 +3
      • 지지선 근처면 +6, 저항 돌파면 +8, 볼린저 상단 돌파는 -3
      • `ATR / 종가 <= 3%` 이면 +5, `>= 6%` 이면 -3
      매도 점수 규칙
      • `EMA20 < EMA50` 이면 +8
      • `EMA50 < SMA200` 이면 +8
      • 종가가 `EMA20` 아래면 +5, `SMA200` 아래면 +5
      • `ADX > 25` 이고 `-DI > +DI` 이면 +4
      • MACD 데드크로스면 +8, 히스토그램 악화면 +4
      • `RSI <= 35` 이면 +6, 과열권 하락 전환이면 +7, `RSI < 25` 이면 -6
      • 하락일에 거래량 비율이 `1.5x` 이상이면 +8
      • 지지선 이탈과 거래량 증가가 동반되면 +7
      • OBV 하락이면 +5, 저항선 근처면 +5, 볼린저 하단 이탈이면 +2
      • `ATR / 종가 >= 4%` 이면 +5
      최종 신호 규칙
      • 매수 점수와 매도 점수 차이가 `10점 미만`이면 `HOLD`
      • 매수 점수가 더 높으면 `BUY`
      • 매도 점수가 더 높으면 `SELL`
      • 강도는 우세 점수 기준으로 계산
      • `80+`: very strong, `65+`: strong, `50+`: watch, 그 미만: weak
      • Point-in-Time과 Backtest는 항상 선택한 날짜까지의 데이터만 사용
